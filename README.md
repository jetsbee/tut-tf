# Tensorflow exercise

## Prerequisites
- Apple silicon machine
- Miniconda

## Create env & install deps
```
$ ./import_conda_env.sh
```

## Activate env
```
$ conda activate tut-tf-env
```

## Deactivate env
```
$ conda deactivate
```

## Check env list
```
$ conda info --envs
```

## Check deps
```
$ conda list
```

## Remove env
```
$ conda remove -n tut-tf-env --all
```

## Export env
```
$ conda env export > environment.yml
```
