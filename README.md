# Tensorflow exercise

## Prerequisites
- Apple silicon machine
- Miniconda

## Create env & install deps
```
$ ./import_conda_env.sh
```

## Activate env
```
$ conda activate ./.conda
```

## Deactivate env
```
$ conda deactivate
```

## Check deps
```
$ conda list
```

## Remove env
```
$ conda remove -p ./.conda --all
```

## Export env
```
$ conda env export > environment.yml
```
